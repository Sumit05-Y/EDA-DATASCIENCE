# 📈 NABIL Bank - Exploratory Data Analysis (EDA)

## 📌 Overview

This project presents an **Exploratory Data Analysis (EDA)** of **NABIL Bank's** one-year historical stock data from the Nepal Stock Exchange (NEPSE). The analysis focuses on understanding the dataset, examining stock market trends, and extracting meaningful insights through statistical analysis.

---

## 📂 Dataset

**Dataset Name:** NABIL Bank One-Year Stock Data

**Source:** Nepal Stock Exchange (NEPSE)

The dataset contains historical trading information, including:

- Trading Date
- Open Price
- High Price
- Low Price
- Closing Price
- Previous Closing Price
- Total Traded Quantity
- Total Traded Value
- Total Number of Transactions

---

## 🛠️ Technologies Used

- Python
- Pandas
- Google Colab

---

## 📊 Analysis Performed

### 1. Data Loading

- Loaded the NABIL stock dataset into a Pandas DataFrame.
- Verified that the dataset was imported successfully.

### 2. Dataset Inspection

- Examined the dataset dimensions.
- Reviewed column names and data types.
- Checked for missing values.
- Identified numerical and categorical features.

### 3. Data Cleaning

- Removed unnecessary columns (if applicable).
- Converted columns to appropriate data types where necessary.
- Ensured the dataset was ready for analysis.

### 4. Statistical Analysis

- Generated descriptive statistics for numerical columns.
- Examined measures such as mean, minimum, maximum, and standard deviation.

### 5. Stock Price Analysis

- Explored trends in opening, closing, highest, and lowest stock prices.
- Compared price movements over the selected period.

### 6. Trading Activity Analysis

- Analyzed trading volume and transaction counts.
- Observed overall trading behavior during the one-year period.

---

## 📈 Key Insights

- The dataset provides historical trading information for NABIL Bank.
- Stock prices fluctuate over time due to market conditions.
- Trading volume varies across different trading days.
- Descriptive statistics provide a summary of the stock's overall performance.
