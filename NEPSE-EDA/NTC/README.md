# 📈 Nepal Telecom (NTC) - Exploratory Data Analysis (EDA)

## 📌 Overview

This project presents an **Exploratory Data Analysis (EDA)** of **Nepal Telecom (NTC)** one-year historical stock data from the Nepal Stock Exchange (NEPSE). The analysis focuses on understanding the dataset, exploring stock price movements, and extracting meaningful insights through statistical analysis.

---

## 📂 Dataset

**Dataset Name:** Nepal Telecom (NTC) One-Year Stock Data

**Source:** Nepal Stock Exchange (NEPSE)

The dataset contains historical trading information, including:

- Trading Date
- Open Price
- High Price
- Low Price
- Closing Price
- Previous Closing Price
- Total Traded Quantity
- Total Traded Value
- Total Number of Transactions

---

## 🛠️ Technologies Used

- Python
- Pandas
- Google Colab

---

## 📊 Analysis Performed

### 1. Data Loading

- Loaded the NTC stock dataset into a Pandas DataFrame.
- Verified successful data import.

### 2. Dataset Inspection

- Examined dataset dimensions.
- Reviewed column names and data types.
- Checked for missing values.
- Identified numerical and categorical features.

### 3. Data Cleaning

- Removed unnecessary columns where applicable.
- Ensured appropriate data types for analysis.
- Prepared the dataset for exploration.

### 4. Statistical Analysis

- Generated descriptive statistics for numerical features.
- Reviewed measures such as mean, minimum, maximum, and standard deviation.

### 5. Stock Price Analysis

- Explored opening, closing, highest, and lowest stock prices.
- Examined price movements throughout the one-year period.

### 6. Trading Activity Analysis

- Analyzed trading volume and transaction counts.
- Observed overall trading activity of NTC shares.

---

## 📈 Key Insights

- The dataset provides historical trading information for Nepal Telecom.
- Stock prices fluctuate over time based on market conditions.
- Trading volume and transaction counts vary across trading sessions.
- Descriptive statistics summarize the overall performance of the stock during the selected period.
