# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Overview

This project presents an **Exploratory Data Analysis (EDA)** of the Titanic dataset using **Python** and **Pandas**. The analysis focuses on understanding the dataset's structure, examining passenger information, and discovering patterns related to passenger survival.

---

## 📂 Dataset

**Dataset Name:** Titanic Dataset

**Source:** Data Science Dojo

The dataset contains passenger information such as:

* Passenger ID
* Survival Status
* Passenger Class
* Name
* Sex
* Age
* Number of Siblings/Spouses Aboard
* Number of Parents/Children Aboard
* Ticket Number
* Fare
* Cabin
* Port of Embarkation

---

## 🛠️ Technologies Used

* Python
* Pandas
* Google Colab

---

## 📊 Analysis Performed

### 1. Data Loading

* Loaded the Titanic dataset directly from its online source.
* Imported the data into a Pandas DataFrame for analysis.

### 2. Dataset Inspection

* Examined the dataset dimensions.
* Reviewed the available columns.
* Checked data types.
* Identified missing values.

### 3. Numerical Data Analysis

* Extracted numerical features for focused analysis.
* Reviewed numerical information available in the dataset.

### 4. Passenger Filtering

* Filtered passengers who survived and were older than 30 years.
* Counted the total number of matching passengers.
* Displayed a sample of the filtered records.

### 5. Survival Analysis by Passenger Class

* Calculated the survival rate for each passenger class.
* Compared survival percentages among First, Second, and Third Class passengers.
* Observed how passenger class influenced survival outcomes.

---

## 📈 Key Insights

* The dataset contains both numerical and categorical features.
* Some columns contain missing values that may require preprocessing.
* Passenger class appears to have a strong relationship with survival.
* Higher-class passengers generally had better survival rates.
* Filtering the dataset helps answer specific analytical questions efficiently.
